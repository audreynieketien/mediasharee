# MediaSharee

A modern photo and media sharing application.

## Features

- User authentication (Consumers and Creators)
- Photo and video uploads for Creators
- Feed with likes and comments
- Search functionality
- Responsive design with Dark Mode support

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
